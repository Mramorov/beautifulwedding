# Beautiful Wedding WordPress Theme

A custom WordPress theme for wedding websites.

## Features

- Custom post type "Svadba" for wedding content
- Custom taxonomy "Places" for organizing wedding venues
- Image gallery support for wedding photos
- Repeater field for additional wedding information
- Responsive design
- Custom navigation menu

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher

## Installation

1. Download the theme
2. Upload it to your WordPress site in wp-content/themes/
