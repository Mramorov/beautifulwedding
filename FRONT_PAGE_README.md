# Шаблон главной страницы Beautiful Wedding

## Обзор

Создан профессиональный шаблон главной страницы для темы Beautiful Wedding с современным дизайном и адаптивной версткой.

## Созданные файлы

### 1. `front-page.php`
Основной шаблон главной страницы WordPress, который автоматически используется для домашней страницы сайта.

**Секции шаблона:**

- **Hero Section** - Приветственный блок с заголовком, описанием и кнопками CTA
- **About Section** - Раздел "О нас" с описанием компании
- **Services Section** - Сетка услуг (отображает записи типа `service`)
- **Weddings Section** - Портфолио свадеб (отображает записи типа `svadba`)
- **Why Us Section** - Преимущества компании (4 карточки с иконками)
- **Contact Section** - Контактная информация и призыв к действию

### 2. `assets/css/front-page.css`
Файл стилей для главной страницы с полной адаптивной версткой.

**Особенности стилей:**
- Использует CSS-переменные темы (цвета, шрифты)
- Responsive дизайн для мобильных устройств
- Анимации при наведении
- Плавная прокрутка к якорям
- Grid-система для карточек

### 3. Обновлен `functions.php`
Добавлена функция автоматической загрузки стилей главной страницы только когда она отображается.

## Использование

### Активация главной страницы

Шаблон `front-page.php` автоматически используется WordPress для главной страницы сайта. Чтобы настроить:

1. Перейдите в **Настройки → Чтение**
2. В разделе "Отображение главной страницы" выберите **Статическая страница**
3. Создайте новую страницу (например, "Главная")
4. Выберите её в качестве главной страницы

### Настройка контента

#### Добавление услуг
Создайте записи типа "Service" в админ-панели. Они автоматически появятся в секции "Наши услуги". Отображаются первые 6 услуг.

#### Добавление свадеб
Создайте записи типа "Svadba" в админ-панели. Последние 3 свадьбы появятся в секции "Наши свадьбы".

#### Изменение контактов
Отредактируйте файл `front-page.php`, найдите секцию `contact-section` и измените:
- Email адрес
- Номер телефона
- Ссылку на анкету

#### Изменение текста "О нас"
Найдите секцию `about-section` в файле `front-page.php` и отредактируйте текст в блоке `about-text`.

### Кастомизация дизайна

Все стили находятся в файле `assets/css/front-page.css`. Основные секции:

- **Hero Section** - Главный баннер
- **Buttons** - Стили кнопок (.btn, .btn-primary, .btn-secondary)
- **Services Section** - Карточки услуг
- **Weddings Section** - Карточки свадеб
- **Features Grid** - Преимущества компании
- **Contact Section** - Контактный блок
- **Responsive Design** - Медиа-запросы для мобильных

## Технические детали

### Зависимости
- WordPress функции: `is_front_page()`, `WP_Query`, `the_post_thumbnail()`
- CSS переменные из `style.css`
- Типы записей: `service`, `svadba`

### Совместимость
- Адаптивный дизайн (desktop, tablet, mobile)
- Современные браузеры (Chrome, Firefox, Safari, Edge)
- Использует CSS Grid и Flexbox

### SEO оптимизация
- Семантическая разметка HTML5
- Правильная структура заголовков (h1-h3)
- Оптимизированные изображения через `the_post_thumbnail()`
- Якорные ссылки для навигации
